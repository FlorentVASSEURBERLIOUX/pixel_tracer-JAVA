Description détaillée

Déclarations des fonctions et structures liées à la gestion des pixels pour différentes formes géométriques.

Ce fichier contient les définitions de la structure Pixel et les déclarations des fonctions permettant de créer et gérer des pixels pour différentes formes géométriques telles que les points, les lignes, les cercles, les rectangles, les carrés, les polygones et les courbes de Bézier.
