Description détaillée

Gestionnaire d'ID global.

Interface pour la gestion des identifiants (ID) uniques.

Ce fichier contient des fonctions permettant de gérer un ID unique global qui peut être utilisé à travers l'application. Les ID sont incrémentés de manière automatique et peuvent être sauvegardés et chargés depuis un fichier.

Ce fichier fournit des déclarations de fonctions pour gérer un identifiant global unique au sein de l'application. L'ID peut être incrémenté automatiquement, et des fonctions permettent de sauvegarder et de charger cet ID à partir d'un fichier.

Note
L'ID est stocké de manière persistante dans le fichier spécifié par ID_FILE.

Documentation des macros

◆ ID_FILE

#define ID_FILE   "id.txt"

Le nom du fichier dans lequel l'ID sera sauvegardé et chargé.

Ce fichier (nommé id.txt) est utilisé pour stocker l'ID global afin qu'il soit persistant entre les sessions de l'application.

Documentation des fonctions

◆ get_next_id()

unsigned long long int get_next_id ( )

Obtient le prochain ID global.

Cette fonction renvoie l'ID suivant en incrémentant une variable globale. Chaque appel à cette fonction génère un ID unique qui peut être utilisé pour identifier des objets ou des entités dans l'application.

Renvoie
Le prochain ID unique sous forme de nombre entier non signé de type long long.
Note
L'ID est incrémenté à chaque appel de cette fonction.

◆ load_id()

void load_id ( )

Charge l'ID global à partir d'un fichier.

Cette fonction charge la valeur de l'ID global depuis un fichier de sauvegarde spécifié par la macro ID_FILE (dans ce cas, "id.txt"). Cela permet de restaurer l'ID utilisé lors des précédentes sessions de l'application.

Note
L'implémentation de cette fonction reste à réaliser.

◆ save_id()

void save_id ( )

Sauvegarde l'ID global dans un fichier.

Cette fonction sauvegarde la valeur actuelle de l'ID global dans un fichier spécifié par la macro ID_FILE (dans ce cas, "id.txt"). Cette opération permet de conserver la valeur de l'ID entre plusieurs exécutions de l'application.

Note
L'implémentation de cette fonction reste à réaliser.

◆ set_id()

void set_id ( unsigned long long int  id)

Définit un ID global spécifique.

Cette fonction permet de définir l'ID global à une valeur spécifique, ce qui peut être utile pour restaurer l'ID à partir d'un fichier de sauvegarde ou pour une initialisation précise.

Paramètres

Note
