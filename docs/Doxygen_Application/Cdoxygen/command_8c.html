Description détaillée

Commande et gestion des paramètres du programme.

Ce fichier contient des fonctions permettant de traiter des commandes de l'utilisateur, de gérer les paramètres associés, et d'interagir avec les objets graphiques dans une application de dessin textuelle basée sur des commandes.

Les principales fonctionnalités incluent la gestion des paramètres de type entier, flottant et chaîne de caractères, ainsi que l'exécution de différentes commandes (par exemple, "exit", "clear", "plot", etc.).
