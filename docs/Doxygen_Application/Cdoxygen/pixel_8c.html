Description détaillée

Gestion des pixels et des formes géométriques.

Ce fichier contient des fonctions pour créer et manipuler des pixels, ainsi que pour convertir diverses formes géométriques en une liste de pixels représentant ces formes à l'écran.

Calcule le point médian entre deux points donnés en fonction du paramètre t.

Cette fonction calcule un point situé entre deux points donnés sur la ligne les reliant, selon une valeur t. Lorsque t = 0, le point est égale à p1, et lorsque t = 1, le point est égale à p2.

Paramètres
p1Le premier point.
p2Le second point.
tLe paramètre de transition (0 <= t <= 1).
Renvoie
Le point médian entre p1 et p2.

< Crée et retourne le point médian
