Description détaillée

Structure représentant une commande avec ses différents paramètres.

Cette structure contient les informations suivantes :

  • name : Le nom de la commande (chaîne de caractères).
  • int_size : Le nombre d'arguments de type entier dans la commande.
  • int_params : Un tableau d'entiers représentant les paramètres de type entier.
  • str_size : Le nombre d'arguments de type chaîne de caractères dans la commande.
  • str_params : Un tableau de chaînes de caractères représentant les paramètres de type chaîne.
  • flt_size : Le nombre d'arguments de type flottant dans la commande.
  • flt_params : Un tableau de flottants représentant les paramètres de type flottant.
